# Агент для прогнозирования финансового состояния предприятий и прогнозирования вероятности банкротства.

Программа принимает 35 входных показателей предприятия (`x1–x35`), прогнозирует 11 обобщенных финансовых блоков и рассчитывает вероятность банкротства. Дополнительно приложение формирует отчет, выделяет слабые и сильные блоки, дает рекомендации и позволяет сохранить PDF-отчет с графиками.

---

## 1. Основные возможности

- Ввод финансовых показателей предприятия вручную;
- Поиск предприятия по ИНН;
- Уточнение поиска по году;
- Случайный выбор предприятия из всего датасета;
- Прогнозирование 11 финансовых показателей предприятия;
- Расчет вероятности банкротства и определение уровня риска;
- Выделение слабых и сильных финансовых блоков;
- Формирование рекомендаций агента;
- Построение PDF-отчета с графиками по текущим прогнозам и истории предприятия.

## 3. Как работает

Программа анализирует финансовое состояние предприятий по 35 входным показателям. Пользователь может ввести значения вручную, выбрать их из списков или загрузить данные по ИНН.

После ввода данные масштабируются и передаются в обученную нейросетевую модель. Модель прогнозирует 11 финансовых показателей предприятия: Reg, Kredit, Teh, Market, Staff, Psich, Ability, Turn, Finn, Z25, Z35.

Затем агент использует эти предсказанные показатели и рассчитывает вероятность банкротства предприятия в процентах. После этого программа определяет уровень риска, выделяет слабые и сильные стороны предприятия, формирует рекомендации и позволяет сохранить PDF-отчет.

В PDF-отчете также строятся графики динамики по выбранному предприятию. График «Динамика финансовой устойчивости» строится по показателям Kredit, Ability, Finn и Z35.
